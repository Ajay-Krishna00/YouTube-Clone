# YouTube Clone

Welcome to my YouTube Clone repository!

## About

This repository contains a clone of the YouTube homepage built using only HTML and CSS. This project is a demonstration of my skills in frontend development and my ability to replicate popular web applications.
### - Website Link: [YouTube Clone](https://ajay-krishna00.github.io/YouTube-Clone/)

- Responsive design for various screen sizes
